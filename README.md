neo_visual_search
=================

Neo4j POC to Integrate VisualSearch.js and Cypher


Instructions
------------

    bundle
    rake neo4j:install[community,2.0.0-M03]
    rake neo4j:start
    rake neo4j:create
    rackup
    open http://localhost:9292
